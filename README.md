# Trabajo final JavaScript Avanzado

Proyecto ficticio desarrollado como trabajo final del módulo.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Ajax con `fetch`
- OpenStreetMap + Leaflet + Leaflet Routing Machine
